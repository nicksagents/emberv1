# AGENTS

This project was scaffolded as a Next.js 15 App Router application.

## Rules

- Keep route code under `src/app/`
- Prefer server components unless interactivity is needed
- Add `"use client"` only where browser state or effects are necessary
- Do not replace the scaffold with a generic UI; preserve the existing visual direction unless the user asks otherwise
- Read `.ember/skills/project-stack/SKILL.md` before making large structural changes
